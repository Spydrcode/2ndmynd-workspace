# Review Rubric

Score each candidate (1–5):

1. **Schema fidelity** — conforms to decision artifact schema.
2. **Doctrine compliance** — avoids dashboard/KPI language and provides finite decision.
3. **Clarity** — calm, grounded, prioritized next steps.
4. **Groundedness** — supported by available facts; no invented metrics.
5. **Helpfulness** — reduces owner decision burden.

Only items with a reviewer + score may be promoted to Growth/Gold.
